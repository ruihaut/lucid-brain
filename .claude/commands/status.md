# /status — where do we stand?

A holistic snapshot for the maintainer. No ceremony.

## Steps

1. Read the current `scrum/sprints/sprint-NN.md` and the shared task-list state.
2. Summarize progress against the sprint goal: what is done, in progress, and blocked.
3. **Reconcile:** compare the live task list against the sprint markdown. Flag any status that is stale or diverging, and treat the markdown as the source of truth.
4. Surface risks and anything that needs a maintainer decision. Keep it short.
