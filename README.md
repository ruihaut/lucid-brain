# Lucid Brain

Application iOS « second cerveau » : transforme ce que tu enregistres au fil du web en connaissance digérée et actionnable. Pas un read-later de plus — un système qui pousse à assimiler et à agir. Le cœur du produit n'est pas la capture, c'est l'après : le « et maintenant ? ».

- **Vision produit** : [VISION.md](VISION.md) (canonique, signée) · [VISION-v1.md](VISION-v1.md) (brainstorm d'origine)
- **Études techniques** : [`ressources/`](ressources/) (pipeline voice-to-text, moteur PKM/note-linking)
